Installation
============

PyHydroGeophysX requires Python 3.8 or higher.

Prerequisites
-------------

Required Dependencies
~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   pip install numpy scipy matplotlib pygimli joblib tqdm

Install from Source
-------------------

.. code-block:: bash

   git clone https://github.com/yourusername/PyHydroGeophysX.git
   cd PyHydroGeophysX
   pip install -e .
